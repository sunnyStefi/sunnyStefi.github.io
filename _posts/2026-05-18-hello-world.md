Welcome. This is the first post.

The site is built with [Jekyll](https://jekyllrb.com) and deployed automatically to GitHub Pages via GitHub Actions whenever a commit lands on `main`.

## Writing a new post

Create a file in `_posts/` named `YYYY-MM-DD-your-title.md` with this front matter:

```yaml
---
layout: post
title: "Your title here"
description: "Optional one-liner shown on the home page."
---
```

Then write in Markdown below the `---`.
